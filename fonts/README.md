# Fonts

Three TrueType fonts from the IM Fell English family by Igino Marini, used by the cover-design scripts in `hpmor-jackets/` (HP fan-fiction book cover generator).

| File | Style |
|------|-------|
| `IMFellEnglish-Regular.ttf` | Regular |
| `IMFellEnglish-Italic.ttf` | Italic |
| `IMFellEnglishSC-Regular.ttf` | Small caps |

Igino Marini's IM Fell revivals are released under the SIL Open Font License — public-domain-equivalent for embedding in printed and digital output. They're checked in here so the cover scripts don't depend on a system font being installed.
